# Nifty 50 Investment Dashboard

An interactive Excel dashboard built using 20 years of historical Nifty 50 stock data. The goal of this project is to help retail investors compare stocks, understand risk, and make more informed long-term investment decisions through data rather than emotions.

---------------------------------------------------------------------------------------------------------------------------

## Problem Statement

Many retail investors rely on news, social media, or market trends while investing, often without analyzing historical performance. This can lead to poor investment decisions and unnecessary risk.

This dashboard addresses that challenge by presenting historical data in a simple and interactive format.

---------------------------------------------------------------------------------------------------------------------------

## Objectives

- Analyze 20 years of Nifty 50 stock data
- Compare stock performance using key financial metrics
- Help investors understand the relationship between risk and return
- Support long-term investment decisions with data-driven insights

---------------------------------------------------------------------------------------------------------------------------

## Tools Used

- Microsoft Excel
- Power Query
- Pivot Tables & Charts
- Slicers
- Excel Formulas

---------------------------------------------------------------------------------------------------------------------------

## Dashboard Highlights

- Investment recommendation based on historical performance
- CAGR analysis
- Risk vs Return comparison
- Maximum Drawdown analysis
- Investment growth projection
- Interactive filters for better exploration

## Key Insights

- Long-term investing has historically delivered better returns than short-term investing.
- Higher returns often come with higher risk, making risk analysis equally important.
- Companies with consistent historical performance generally provide more stable investment opportunities.
- Visualizing historical data makes it easier to compare stocks and identify suitable investment options.

---------------------------------------------------------------------------------------------------------------------------

## Impact

This dashboard helps retail investors move beyond guesswork by providing a simple way to compare stocks using historical data. Instead of relying on market noise, users can evaluate investments based on risk, return, and long-term performance.

---------------------------------------------------------------------------------------------------------------------------

## How to Use

1. Open the Excel dashboard.
2. Select the desired filters.
3. Explore the charts and KPIs.
4. Compare stocks using CAGR, risk, and return metrics.
5. Use the insights to support long-term investment decisions.
